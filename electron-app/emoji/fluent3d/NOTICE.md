# Fluent Emoji (3D)

3D emoji from **Microsoft Fluent UI Emoji** — https://github.com/microsoft/fluentui-emoji

- License: **MIT** — Copyright (c) Microsoft Corporation.

**Curated subset (~172 PNGs)** — the avatar's face vocabulary: the Unicode
"Smileys & Emotion" group + emojis bots have actually used (from call logs) + a
few common icons (👀 🎉 ❤️ …). Files are PNG named by the canonical hex codepoint
(same scheme as the other sets). Fluent ships no default-skin hand/person
gestures (🙋 👋 👍 …), so those fall back to the native glyph.

Re-fetch/re-curate with `scripts/fetch-fluent3d.mjs`.
